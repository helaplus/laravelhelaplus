## Installation

Via Composer

``` bash
$ composer require helaplus/laravelhelaplus
```

## Usage
Add the following Configs
``` bash
B2B_INITIATOR
B2B_PASSWORD
B2B_SOURCE
B2B_RESULT_URL
B2B_CALLBACK_URL
```
